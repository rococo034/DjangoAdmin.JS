# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.2.0] - 2026-06-13
### Added
- Premium Drag-and-Drop File Uploader with real-time preview, support for both new file uploads and existing file deletion.
- Animated, real-time interactive charts on the dashboard.
- Redesigned command palette search UX.
- Custom theme customization engine (classic, glassmorphism, minimalist).
- Library version display and repository link.

## [0.1.0] - 2026-06-12
### Added
- Initial release of DjangoAdmin.JS.
- Redesigned Django Admin templates with Tailwind CSS.
- Modern dashboard with statistics.
- Responsive navigation and layout.
- Dark mode support.
- Professional package structure with `pyproject.toml`.
- GitHub Actions for CI/CD.
