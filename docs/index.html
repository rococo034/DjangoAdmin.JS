Quick Installation

Upgrade your Django workspace in minutes.

1. Install DjangoAdmin.JS via pip

pip install django-admin-js

2. Register in your INSTALLED_APPS

Ensure it is ordered before Django's core administration application:

INSTALLED_APPS = [
    "django_admin_js",  # Add this before Django Admin
    "django.contrib.admin",
    "django.contrib.auth",
    ...
]
